# Skills Directory

이 디렉터리에는 공개 스펙 위에서 만들어지는 설치형 또는 레퍼런스 스킬 패키지가 들어간다.

현재 첫 항목:

- `decision-core`
- `decision-capture`

이후 워크플로별 스킬 후보:

- `ci-rationale`
- `pr-rationale`
- `handoff-context`
- `memory-promote`

현재 추가된 초기 skill은 보안 원칙에 따라 instruction-first, non-executable 구조를 유지한다.
현재 포함된 자동화는 좁은 범위의 로컬 도구뿐이다. `decision-core`에는 schema validation, `decision-capture`에는 draft record scaffolding이 들어 있다.
