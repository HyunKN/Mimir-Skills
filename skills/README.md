# Skills Directory

This directory will hold installable or reference skill packages built on top of the public specs.

Current first entries:

- `decision-core`
- `decision-capture`

Future workflow-specific skills may include:

- `ci-rationale`
- `pr-rationale`
- `handoff-context`
- `memory-promote`

These initial skills are instruction-first and non-executable by design.
They currently include only bounded local automation: schema validation in `decision-core` and draft record scaffolding in `decision-capture`.
