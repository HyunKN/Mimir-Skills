# Skills Directory

This directory will hold installable or reference skill packages built on top of the public specs.

Current first entries:

- `decision-core`
- `decision-capture`
- `dependency-upgrade-decision`
- `ci-rationale`
- `handoff-context`
- `pr-rationale`

Future workflow-specific skills may include:

- `memory-promote`

These initial skills are instruction-first and non-executable by design.
They currently include only bounded local automation: schema validation in `decision-core`, draft record scaffolding and summary rendering in `decision-capture`, and workflow-specific guidance in `dependency-upgrade-decision`, `ci-rationale`, `handoff-context`, and `pr-rationale`.
