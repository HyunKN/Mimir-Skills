# Evaluations Directory

이 디렉터리에는 문서와 record가 실제로 유용한지 확인하기 위한 evaluation 프롬프트와 시나리오가 들어갈 예정이다.

초기 평가 주제:

- 저장된 record만으로 새 에이전트가 이어서 작업할 수 있는가
- 리뷰어가 결정을 빠르게 이해할 수 있는가
- 승격된 memory가 현재 저장소 상태에서도 여전히 유효한가

현재 공개 평가 문서:

- `replay-evaluation.ko.md`
- `reviewer-comprehension.ko.md`

현재 평가는 다음 공개 가능한 합성 example을 기준으로 한다.

- `../examples/windows-ci-timeout/`
- `../examples/cache-client-pin/`
